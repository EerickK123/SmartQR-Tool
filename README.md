# SmartQR Tool

SmartQR Tool is a QR code generation system designed to improve product traceability by embedding structured product and batch information into QR codes.

## Project Status
Design phase

## Main Features
- QR generation with product and lot information
- Role-based access (admin, supervisor, operator)
- Action history and traceability
- Support for batch and unit identification

## Tech Stack (planned)
- Backend: Node.js + Express + TypeScript
- Frontend: React + TypeScript
- Database: PostgreSQL (or SQLite for local)
